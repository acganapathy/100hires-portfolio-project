 # 100Hires Portfolio Project – Setup Documentation

## Objective

The objective of this project was to demonstrate my ability to independently learn and use new tools, troubleshoot technical issues, and document my work clearly. This project involved setting up development tools, creating a GitHub repository, working within Cursor IDE, and publishing documentation online.

## Tools Installed

- Cursor IDE
- Claude Code Extension (Cursor)
- Codex Extension (Cursor)
- GitHub
- Apple Command Line Developer Tools (required for Git on macOS)

## Steps Completed

### Step 1: Installed Cursor IDE

- Downloaded and installed Cursor IDE on my MacBook.
- Created an account and logged in successfully.

### Step 2: Installed Claude Code Extension

- Opened Extensions inside Cursor.
- Searched for "Claude Code".
- Installed the extension and logged in.

### Step 3: Installed Codex Extension

- Opened Extensions inside Cursor.
- Searched for "Codex".
- Installed the extension and logged in.

### Step 4: Created a Public GitHub Repository

- Created a repository named "100hires-portfolio-project".
- Enabled public visibility so it could be shared with reviewers.

### Step 5: Opened the Repository in Cursor

- Connected the GitHub repository inside Cursor.
- Opened the project workspace successfully.

### Step 6: Created README.md Documentation

Created a README.md file that included:

- Project objective
- Tools installed
- Steps completed
- Challenges encountered
- Solutions implemented
- Skills demonstrated

### Step 7: Installed Apple Developer Tools

During Git operations, Git was unavailable because macOS Command Line Developer Tools were not installed.

Action taken:

- Installed Apple Command Line Developer Tools.
- Verified Git functionality afterwards.

### Step 8: Git Version Control

Used Git commands to upload the project to GitHub.

Commands executed:

git add README.md

git commit -m "Update README"

git push

The repository was successfully updated online.

## Challenges Encountered

### Challenge 1: Understanding GitHub and Cursor Integration

Initially, I was unfamiliar with how Cursor and GitHub work together.

Solution:

- Researched tutorials and learned the workflow between GitHub repositories and Cursor IDE.

### Challenge 2: Git Not Working on macOS

When attempting to use Git, macOS displayed:

"No developer tools were found."

Solution:

- Installed Apple's Command Line Developer Tools.

### Challenge 3: GitHub Authentication

Git requested authentication during the push process.

Solution:

- Logged in using my GitHub account and completed the authentication process successfully.
